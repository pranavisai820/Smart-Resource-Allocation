# Smart Resource Allocation System

A web-based platform to help NGOs and social organizations identify urgent community needs and efficiently assign volunteers based on priority and skills.

## Problem Statement
Community data is often scattered across surveys and reports, making it difficult to identify urgent issues and allocate resources effectively.

## Solution
This system collects community needs, prioritizes them based on severity, and matches volunteers based on their skills and availability.

## Features
- Add community needs
- Add volunteers
- Smart allocation based on priority
- Priority levels (High, Medium, Low)
- Automatic volunteer assignment
- Clean and user-friendly interface

## Tech Stack

- CSS
- JavaScript
- Python (Flask)
